!!!! WARNING !!!!

After executing this application will be created files image1.mcs and image2.mcs.
If they exist, they will be overwritten !!!

Procedure:

1) Create a bitmap with dimensions 156x177 or 160x229.
    This bitmap should contain only 2 colors: black(0x000000) and white (0xFFFFFF).
    All non-black colors will be converted to white color.

2) Select desired size in the combobox.

3) Open created bitmap by clicking the button [Open BMP]

4) Set position ( 1..40 )

5) For saving image press the [save] button.

6) Points 2..5 could be repeated, if you want to.

This application generates files:
image1.mcs ... 40 PICT-images 156x177
image2.mcs ... 40 PICT-images 160x229
These files could be opened by CPmanager and transfered to CP300.
